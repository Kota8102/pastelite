AWSのEC2でセキュリティグループを設定しようとした時、自分のPCのIPを登録しようとしたのですが、Zscalerを利用しているため、IPが可変であるため、登録に困りました。\
そこで、ZscalerのIPについて調べてみることにしました。

## Zscalerとは

Zscalerは、クラウドベースのセキュリティ製品です。
[ゼロトラスト](https://www.zscaler.jp/zero-trust)のセキュリティモデルに基づいており、多くの企業でのセキュリティ対策として採用されています。
Zscalerを利用すると、ユーザーのトラフィックはZscalerのプロキシサーバーを経由するため、出力されるIPアドレスはZscalerのものとなります。

## ZscalerのIPアドレス (Location: Tokyo IV)
以下の4つのIPアドレスがZscalerのIPアドレスとして登録されていました。(2023/08/16現在) \

1. **165.225.110.0/23**

2. **165.225.96.0/23**

3. **147.161.198.0/23**

4. **136.226.238.0/23**

上記のIPをセキュリティグループのインバウンドルールに設定しておくとアクセスすることができます。

### 注意事項
これらのIP範囲は、[Zscalerサービス継続性ポリシー](https://help.zscaler.com/zia/zscaler-service-continuity-customer-notification-protocol)に従い、60日間公開された後にいつでも使用開始される可能性があります。\
そのため、以下のリンクから最新の情報を確認してください。\
[Cloud Enforcement Node Ranges](https://config.zscaler.com/zscaler.net/cenr)
